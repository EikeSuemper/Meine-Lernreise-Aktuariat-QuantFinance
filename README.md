# Meine-Lernreise-Aktuariat-QuantFinance
Hier befindet sich meine Lernreise im Bereich der quantitativen Finanzen und des Aktuariats. Es dient als Dokumentation meines Lernfortschritts sowie als Portfolio meiner Fähigkeiten.

# 01_Grundlagen
In diesem Ordner geht es um Grundlagen bezüglich der Anwendung von Programmen. Ich bilde mich aktiv über DataCamp fort und absolviere Zertifikate, die mir den Berufseinstieg erleichtern sollen. Dazu gehört aktuell Python. Längerfristig werde ich mich ebenso mit R-Studio und SQL auseinandersetzen. 

# 02_Aktuariat
Der Ordner umfasst sowohl theoretische Erkenntnisse aus meinem Studium, als auch praktische Übungen (Projekte), in denen ich das theoretisch erlernte Wissen umsetze. Hierzu zählen ebenso eigenständig erlernte Methoden und Modelle aus der Versicherungsmathematik, sowie Einflüsse aus Arbeit/Praktika/Webinare/Bücher.
