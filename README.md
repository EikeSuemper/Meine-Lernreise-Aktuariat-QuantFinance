# 📈 Meine Lernreise: Aktuariat & Quantitative Finance

Dieses Repository dient als technisches Portfolio und dokumentiert meinen kontinuierlichen Lernfortschritt im Bereich Aktuariat, Risikomanagement und Quantitative Finance.
Ziel ist es, meine Eigeninitiative, mein analytisches Denken sowie meine Fähigkeit zu zeigen, mathematische Modelle aus Studium, Praktika und eigener Weiterbildung praktisch zu implementieren.

---

## 🎯 Zielsetzung

* Kontinuierliche Weiterbildung in quantitativen Finanz- und Versicherungsthemen
* Umsetzung theoretischer Inhalte in praktische Pythonimplementierungen
* Aufbau eines nachvollziehbaren, strukturierten Quant-/Aktuariats-Portfolios
* Entwicklung eines tiefen Verständnisses für Modelle, Methoden und Daten

---

## 🧠 Struktur des Repositories

Das Repository ist in mehrere zentrale Bereiche unterteilt:

### 📚 Grundlagen

Hier stehen das Erlernen von Tools und Methoden im Fokus.
* Einführung in Python
* Erklärung zentraler Funktionen und Konzepte
* Datenanalyse und Visualisierung
* Mathematische Grundlagen in Code

---

### 💼 Aktuariat

Dieser Bereich behandelt Themen der Versicherungsmathematik.

* Prämienkalkulation
* Schadenmodelle

* Regulatorische Themen
* Umsetzung aktuarieller Modelle in Python

### 📈 Quantitative Finance

Fokus auf finanzmathematische Modelle und deren Implementierung.

* Optionspreismodelle (z. B. Black-Scholes)
